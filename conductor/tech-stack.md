# Tech Stack

## Overview
This document outlines the core technologies and frameworks used in the CouponCupid project.

## Core Technologies

*   **Backend:** Node.js with Express.js
    *   **Description:** Express.js is a minimal and flexible Node.js web application framework that provides a robust set of features for web and mobile applications. It's used here to build the server-side logic, handle API requests, and manage data interactions.

*   **Frontend:** Vue.js
    *   **Description:** Vue.js is a progressive JavaScript framework for building user interfaces. It is used to create the interactive and dynamic client-side application for CouponCupid, providing a responsive and engaging user experience.

*   **Database:** PostgreSQL
    *   **Description:** PostgreSQL is a powerful, open-source object-relational database system. It serves as the primary data store for CouponCupid, managing product information, coupon data, user lists, and other essential application data.

*   **Styling:** Bootstrap
    *   **Description:** Bootstrap is a free and open-source CSS framework directed at responsive, mobile-first front-end web development. It is utilized to ensure a consistent, visually appealing, and responsive design across the CouponCupid application.
